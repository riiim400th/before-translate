# BeforeTransrate

## 説明
ページ翻訳前にこの拡張機能を実行することで、翻訳の邪魔になるcodeタグが除かれ、自然な翻訳になります

## インストール

リポジトリをクローン 
```bash
git clone https://github.com/riiim400th/before-translate.git
```

`chrome://extensions/` からデベロッパモードをON

`パッケージ化されていない拡張機能を読み込む`からクローンしたディレクトリを選択

## 実行

1. コンテキストメニュー(右クリック)から「日本語に翻訳する前に...」を選択

2. コンテキストメニュー(右クリック)から「日本語に翻訳する」を選択
